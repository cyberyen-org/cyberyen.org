# About

Cyberyen.org is an [open-source](https://github.com/cyberyen-org/cyberyen.org) website, open to contributions from any developer. All regular activity is organized through a public [pull request](https://github.com/cyberyen-org/cyberyen.org/pulls) process and managed by the site co-maintainers.

**Just as nobody owns the Internet, nobody owns Cyberyen.**

## Some Rules

Respect the following rules when when contributing to cyberyen.org:

* Do not submit copyrighted work without permission.
* Use the [#cyberyen.org](https://matrix.to/#/#cyberyen.org:matrix.org) Matrix to resolve disputes before engaging in edit wars on any particular article.
* Provide proper attribution when adding content from sources that require it.
* No advertisements or spam.
* Do not include donation addresses in the content you contribute.

You can report any problem or help to improve cyberyen.org on [GitHub](https://github.com/cyberyen-org/cyberyen.org) by opening an issue or a pull request in English. When submitting a pull request, please take required time to discuss your changes and adapt your work. Please don't ask for promotion for your personal business or website, except for special cases like conferences. Many thanks to all contributors who are spending time improving Cyberyen!
